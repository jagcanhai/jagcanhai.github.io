1. Visual Studio Code


Visual Studio Code 是一个免费、轻量级且可扩展的代码编辑器，用于使用任何编程语言和框架构建 Web、桌面和移动应用程序。支持 JavaScript 和 TypeScript，并提供用于 Python、Java、C/C++、C#、Go、Rust、PHP 等语言的编码扩展。

Visual Studio Code官网

2. IntelliJ IDEA


IntelliJ IDEA是JetBrains开发的一款Java集成开发环境，支持Java以及Kotlin、Scala、Groovy等JVM语言开发。

IntelliJ IDEA官网

3. PyCharm


PyCharm由JetBrains出品，是一款专门为Python开发设计的IDE。适用于Python开发，包括Web开发、数据科学和机器学习等。

PyCharm官网

4. Netbeans


NetBeans 是一个免费的、开源的集成开发环境，适用于 Windows、Mac、Linux 和 Solaris 操作系统上的应用程序开发。该 IDE 简化了使用 Java 和 HTML5 平台的 Web、企业、桌面和移动应用程序的开发。

netbeans官网
